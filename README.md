### Installation

- Just, Clone this repository -
```
$ git clone git://github.com/htr-tech/zphisher.git
```

```
$ bash Phins.sh
```

### Dependencies

**`Phins`** requires following programs to run properly - 
- `php`
- `wget`
- `curl`
- `git`

> Supported Platform : **`Termux`**, **`Ubuntu/Debian/Kali/Parrot`**, **`Arch Linux/Manjaro`**, **`Fedora`**
